# 移动通信


## 20222 学期 刘赟宇


### 简介

程立足移动通信的最新发展和技术选取教学内容，以移动通信的喜闻乐见应用场景为切入点，结合生活实际，讲解从第一代移动通信系统到第五代移动通信系统的系统、关键技术等，利用虚拟仿真软件进行网络规划与建设。
