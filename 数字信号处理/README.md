# 数字信号处理

## 20221  学期 刘涛

### 简介

数字信号处理（Digital signal processing），简称DSP。其目的是对真实世界的模拟信号进行加工和处理。因此在数字信号处理前，模拟信号要用模数转换器（A-D变换器）变成数字信号；经数字信号处理后的数字信号往往要用数模转换器（D-A变换器）变回模拟信号，才能适应真实世界的应用。
